## About constellation
Constellation is a framework for realtime, peer to peer distributed applications, with a focus on resiliance and security. 

## WebAssembly Demo

Try the live WebAssembly demo:
- Main: [https://Stormancer.github.io/constellation/](https://Stormancer.github.io/constellation/)
- Develop: [https://Stormancer.github.io/constellation/develop/](https://Stormancer.github.io/constellation/develop/)

The `main` branch deploys to the root, `develop` to `/develop/`, and tags to `/tagname/`.
